# Telegram Bot (aiogram + dotenv)

Простой Telegram‑бот на Python с использованием `aiogram` и загрузкой токена из `.env`.

## Возможности

- Команда `/start` — приветствие.
- Команда `/help` — справка по командам.
- Эхо‑режим: повторяет любой текст пользователя.

## Установка и запуск

1. Клонируй репозиторий.

2. Создай и активируй виртуальное окружение:

   ```bash

   python -m venv venv

   .\venv\Scripts\Activate.ps1

3. Установи зависимости:

bash

pip install -r requirements.txt

4. Создай файл .env в корне проекта и добавь туда:

env

BOT_TOKEN=твой_токен_от_BotFather

5. Запусти бота:

bash

python bot.py

6. Особенности реализации

Токен хранится отдельно в .env и не попадает в репозиторий (защищён через .gitignore).
Используется асинхронный стек aiogram 3.x.
