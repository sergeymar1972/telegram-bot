# Telegram Bot (aiogram + dotenv)

Простой Telegram‑бот на Python с асинхронной обработкой сообщений и встроенным калькулятором.

## Возможности

- Команда `/start` — приветствие и кнопка «Калькулятор».
- Команда `/help` — справка по командам.
- Команда `/calc <выражение>` — вычисление арифметического выражения.
- Команда `/date` — возвращает текущую дату русском языке и время.
- Кнопка «🧮 Калькулятор» — запрос на ввод выражения.
- Безопасное вычисление: запрещены опасные операции, проверяется набор символов.
- Обработка ошибок: деление на ноль, недопустимые символы и т.п.

## Тесты

Проект содержит тесты для функции safe_eval, проверяющие:

корректные арифметические выражения;

деление на ноль;

недопустимые символы и попытки выполнения опасного кода;

пустые и пробельные выражения;

тест для функции cmd_date, проверяющий

формат даты

## Особенности реализации

Токен хранится отдельно в .env и не попадает в репозиторий (защищён
 через .gitignore).

Используется асинхронный стек aiogram 3.x.

Вычисления выполняются в изолированном окружении без доступа к встроенным функциям Python.
